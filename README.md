# INTRODUCTION
This repository was created as part of my class project for Intro to Server-Side Scripting + PHP. For the project, I was to build a simple PHP web application that allowed users to select a quantity of an item(s) to purchase, add to their shopping cart, and checkout. 

# FEATURES
**Catalog Page:** Displays all available products
**Cart Page:** Allows users to add, remove, update quantities of items
**Checkout Page:** Shows a detailed order summary - tax, subtotal, shipping
**Confirmation Page:** Clears the cart and confirms orders
**MVC Structure:** Separates logic, data, and presentation for maintainability

# TECHNOLOGIES USED

**MySQL / phpMyAdmin**
**HTML5**
**XAMPP**
